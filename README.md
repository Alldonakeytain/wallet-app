# Консольный Банк (Wallet App)

Приложение на Python 
Программа работает в консоли и симулирует простой банк с базой данных.

## Что умеет программа:
- Можно зарегистрировать нового пользователя или войти под существующим именем.
- Можно пополнять баланс и снимать деньги со счета.
- Есть функция перевода денег другому пользователю (баланс обновляется у обоих сразу).
- Программа сохраняет историю всех действий (пополнения, снятия, переводы) с датой и временем.

## Технологии:
- Python 3
- SQLite3 (все данные пользователей и история хранятся в файле `bank.db`)

## 🚀 Как запустить у себя:
1. Скачайте код к себе на компьютер.
2. Откройте терминал в папке с проектом и запустите файл `main.py`:
   ```bash
   python main.py```
